Thank you for participating in our experiment!

The goal of the game is to control a spacecraft with your keyboard to land it on the platform. Each day, you will have a 20-minute session. After each game session, we will ask you to answer some questions. You may have to fill out the same questionnaire several times. Try to answer it as accurately as possible.

You can control the spacecraft with any of the three action buttons:

  • J tilts the spacecraft clockwise
  • F tilts the spacecraft counter-clockwise
  • Space thrusts spacecraft ahead
{% include "JOLD/tutorial_ll.html" %} Additionally, you will see two on-screen buttons:
  • Fullscreen switches the fullscreen mode on and off. The game can only be played with fullscreen activated.
  • Terminate terminates current session. If you terminate the session before the time runs out, we will not be able to use your data in our study.
