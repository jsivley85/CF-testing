This is a demo tab written in basic html. You can use these tabs to present instructions or provide any other information that the user needs to know before continuing. The tabs for this page are located in Flowers_OL/flowers-ol/demo_app/templates. The html files are linked with the task home page through the Task object which has the info_templates_csv field. This field has a specific syntax which is parsed by Django through the home_page.html (Flowers_OL/flowers-ol/manager_app/templates).
