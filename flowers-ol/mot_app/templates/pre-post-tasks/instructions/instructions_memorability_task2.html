Instructions

  • The goal of this experiment is to measure the ability of memory capacity for photographs.
  • In this experiment, you will see a set of photographs one-by-one.
  • Your task is to remember each photograph and answer whether the photograph is presented twice or not by pressing the key "f" as soon as possible.
  • Some images will appear twice in a short-range, others in a long-range, and the others just once.
  • In each trial, you will get feedback as to if your response was correct.
    • If you answer correctly (that is, if you don't press the key for the first-time-image or if you do press the key for the image presented twice), you will see the word "Correct answer".
    • If not, you will see the word "Wrong answer. "
  • You can check the practice trials of this task from the following window.
